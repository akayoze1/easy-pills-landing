The Challenge

The Adherence Crisis

Medication non-adherence accounts for nearly 50% of treatment failures and 125,000 deaths annually. The inability to track, verify, and dispense medication securely creates a gap in the healthcare continuum.

The Solution

Intelligent Automation

Easy Pills creates a closed-loop system. By combining biometric authentication with automated dispensing motors and cloud logging, we eliminate human error and unauthorized access.

  • Zero-Trust Biometric Access

02

Technical Specifications

A comprehensive overview of the hardware and software subsystems powering the device.

Biometric Identity

Optical fingerprint sensor enabling secure multi-user authentication.

The AS608 optical fingerprint module captures fingerprint images and stores up to 1000 fingerprint templates in onboard flash memory. It communicates with the ESP32 via UART to provide authorized access control.

Robotic Dispensing

28BYJ-48 stepper motors provide precise rotational control for accurate pill positioning and dispensing.

Two 28BYJ-48 stepper motors are controlled by the ESP32 through ULN2003 driver boards. Each motor provides 4096 steps per revolution with gear reduction, enabling accurate positioning of the medication compartments and controlled pill release.

Cloud Infrastructure

Firebase Firestore integration for secure synchronization of medication logs and schedules.

The cloud backend uses Firebase Firestore as the primary database platform to synchronize medication schedules, dispensing logs, and user data between the device, mobile application, and doctor monitoring portal.

Security Protocols

Digital locking mechanisms combined with biometric and password-based access control.

The system uses fingerprint authentication and a password-protected electric lock to prevent unauthorized access. The access code changes every 12 or 24 hours and is sent to the owner, providing an additional layer of physical security.

Smart Notifications

Local audiovisual alerts synchronized with mobile push notifications.

The notification system combines an active buzzer for audio alerts with LED status indicators for visual notifications.

Inventory Intelligence

Automated tracking of dispensed doses and expiration dates through cloud logging.

The system tracks medication dispensing events and expiration dates using cloud-based logs. Each dispensing operation is recorded with the medication type, quantity, and timestamp to monitor adherence and inventory status.
