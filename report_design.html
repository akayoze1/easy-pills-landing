Core Functionality
Feature Overview
🔐
Authentication & Security
  • Email/password login with Firebase Auth
  • Role-based access: Patient vs. Doctor
  • Password reset via email link
  • Biometric fingerprint on hardware device
  • Enrolled user roles: Owner, Family, Caregiver
  • Full access audit log with timestamps
💊
Medication Management
  • 6 physical compartments (C1–C6)
  • Add, edit, delete medication schedules
  • Group doses by time of day
  • Toggle schedules active/inactive
  • Real-time countdown to next dose
  • Doctors can manage patient schedules remotely
📦
Inventory Tracking
  • Pill count and capacity per compartment
  • Visual stock level progress bar
  • Low-stock alert when <20% remaining
  • Expiry date tracking per compartment
  • Expiring-soon warning (<60 days)
  • Tap to edit, long-press to clear slot
📊
Adherence Monitoring
  • Real-time taken/total counter per day
  • 7-day visual dot heatmap per patient
  • Weekly adherence percentage
  • Consecutive-day streak tracking
  • Color-coded status (≥80% green, ≥60% amber)
  • Doctor-wide aggregate adherence score
👥
Doctor–Patient Linking
  • Patient shares unique Patient ID
  • Doctor adds patients by entering ID
  • Patient can unlink doctor at any time
  • Doctor views all assigned patients
  • Missed-dose alerts for at-risk patients
  • Patients grouped: missed / on-track / no activity
📡
Hardware Integration
  • 5-step WiFi device pairing wizard
  • 2.4 GHz WiFi connection via BLE setup
  • Live device online/offline status indicator
  • Battery level monitoring
  • "Dispense Now" button triggers hardware
  • Physical fingerprint sensor on dispenser
